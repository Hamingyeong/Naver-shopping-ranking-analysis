# 네이버 쇼핑 상위노출 요인 분석

## 1. 프로젝트 개요

본 프로젝트는 네이버 쇼핑 검색 결과에서 상품이 상위에 노출되는 패턴을 데이터마이닝 관점에서 분석한 프로젝트입니다.
분석 대상은 키보드와 의자 카테고리이며, 개별 상품 자체가 아니라 **상품 × 검색 키워드 관측치**를 분석 단위로 설정했습니다.

본 분석의 목적은 특정 상품이 상위노출되는 “공식”을 규명하는 것이 아니라, 리뷰수, 가격, 브랜드 관심도, 배송 정보, 제목-키워드 매칭률 등 여러 feature가 상위노출 여부와 어떤 관련 패턴을 보이는지 탐색하는 데 있습니다.

## 2. 분석 목적

본 프로젝트의 주요 분석 목적은 다음과 같습니다.

1. 네이버 쇼핑 검색 결과에서 Top38 및 Top100 상위노출 상품을 예측한다.
2. 상위노출 여부와 관련된 주요 feature 패턴을 파악한다.
3. Top100 후보군을 먼저 선별한 뒤, 그 내부에서 Top38 상품을 재랭킹하는 2단계 모델 구조를 실험한다.
4. 리뷰수 관련 변수를 제외했을 때, 가격, 배송, 브랜드, 제목 관련 변수의 중요도가 어떻게 달라지는지 비교한다.
5. 검색 키워드별로 상위노출 예측 성능이 어떻게 달라지는지 확인한다.

## 3. 데이터 설명

분석에는 네이버 쇼핑 검색 결과 데이터와 브랜드 관심도 관련 데이터를 활용했습니다.

주요 데이터 항목은 다음과 같습니다.

* 상품명
* 검색 키워드
* 네이버 쇼핑 노출 순위
* 가격
* 리뷰 수
* 평점
* 브랜드 여부
* 브랜드 관심도
* 배송 관련 정보
* 제목과 검색 키워드의 매칭 정도
* 카테고리 정보

분석에서 사용한 target variable은 다음과 같습니다.

* `top38`: 검색 결과에서 organic rank가 38위 이내이면 1, 아니면 0
* `top100`: 검색 결과에서 organic rank가 100위 이내이면 1, 아니면 0
* `rank_score`: 검색 키워드별 노출 순위를 정규화한 상대적 순위 점수

## 4. Raw data 미포함 안내

본 저장소에는 원천 데이터(raw data)를 포함하지 않았습니다.

원천 데이터는 네이버 쇼핑 검색 결과 및 외부 서비스 기반 데이터로 구성되어 있으며, 데이터 출처와 재배포 제한, 수집 환경 차이, API 인증 정보 문제로 인해 GitHub 저장소에 공개하지 않았습니다.

따라서 본 저장소는 원천 데이터 자체보다는, 분석에 사용한 모델링 코드와 데이터마이닝 절차를 확인할 수 있도록 구성했습니다.

## 5. 분석 방법

본 프로젝트에서는 supervised learning 기반의 분류 및 회귀 모델을 사용했습니다.

### 5.1 분류 모델

상위노출 여부를 예측하기 위해 다음 모델을 사용했습니다.

* Logistic Regression
* Random Forest
* LightGBM

분류 target은 `top38`과 `top100`입니다.

평가 지표는 다음을 사용했습니다.

* Precision
* Recall
* F1-score
* ROC-AUC
* Average Precision
* Precision@38
* Recall@38
* NDCG@38

### 5.2 2단계 재랭킹 구조

Top38은 전체 관측치 중 비율이 낮은 희소한 target이므로, 직접 예측뿐 아니라 2단계 구조도 실험했습니다.

1단계에서는 Top100 후보군을 예측하고,
2단계에서는 1단계에서 선별된 후보군 내부에서 Top38 여부를 다시 예측했습니다.

수정된 최종 코드에서는 후보군 선별 과정의 검증 편향을 줄이기 위해 OOF(out-of-fold) 방식으로 1단계 예측 확률을 생성했습니다.
즉, 각 관측치는 자기 자신을 학습하지 않은 모델의 예측값을 기반으로 후보군에 포함되도록 처리했습니다.

### 5.3 회귀 모델

상위노출 여부뿐 아니라 순위의 상대적 정도를 보기 위해 `rank_score`를 target으로 하는 회귀 분석도 수행했습니다.

사용 모델은 다음과 같습니다.

* Linear Regression
* Ridge Regression
* Random Forest Regressor
* LightGBM Regressor

회귀 분석은 Top38 분류 결과를 보완하기 위한 보조 분석으로 사용했습니다.

## 8. 주요 분석 결과 요약

분석 결과, 리뷰수 관련 변수는 키보드와 의자 카테고리 모두에서 상위노출과 강하게 연결된 패턴을 보였습니다.

다만 리뷰수는 상위노출의 원인이라기보다, 이미 많이 노출된 상품에 리뷰가 더 많이 쌓이는 내생성 문제를 포함할 수 있습니다. 따라서 본 분석에서는 리뷰수 관련 변수를 제외한 추가 모델도 함께 비교했습니다.

리뷰수 관련 변수를 제외했을 때는 카테고리별로 다른 feature가 중요하게 나타났습니다.

* 키보드 카테고리: 가격, 제목-키워드 매칭률, 배송 정보 등이 주요 변수로 나타남
* 의자 카테고리: 브랜드 관심도, 가격, 배송 및 카테고리 정보 등이 주요 변수로 나타남

또한 Top100 후보군을 먼저 선별한 뒤 Top38을 재랭킹하는 2단계 구조는 후보군 내부에서 상위노출 상품을 구분하는 데 유용한 가능성을 보였습니다.

## 9. 한계점

본 분석에는 다음과 같은 한계가 있습니다.

1. 네이버 쇼핑 랭킹 알고리즘 전체를 알 수 없으므로, 본 분석은 실제 알고리즘을 복원한 것이 아니라 관찰된 데이터 기반 패턴 분석입니다.
2. 리뷰수는 상위노출의 원인이면서 동시에 결과일 수 있어 내생성 문제가 존재합니다.
3. 수집 시점과 검색 환경에 따라 네이버 쇼핑 순위는 달라질 수 있습니다.
4. raw data를 공개하지 않았기 때문에 저장소만으로 완전한 재현은 어렵습니다.
5. 본 분석은 상품-키워드 관측치 단위의 패턴 분석에 초점을 두었으며, 완전히 새로운 상품에 대한 일반화 성능은 추가 검증이 필요합니다.

## 10. 추후 개선 방향

추후에는 다음과 같은 방향으로 분석을 확장할 수 있습니다.

1. 수집 시점을 늘려 시간에 따른 순위 변동 분석
2. 더 많은 카테고리와 검색 키워드로 분석 범위 확장
3. 동일 상품 ID 기준 GroupKFold 검증을 통한 신규 상품 일반화 성능 확인
4. 클릭률, 구매 전환율, 광고 여부 등 추가 변수 확보
5. 후보군 선별과 재랭킹을 포함한 end-to-end ranking model 고도화
